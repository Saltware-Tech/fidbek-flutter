# fidbek_flutter v0.2.0

## Summary
Native SDK refresh and reliability release.

## Changed
- Updated bundled Android artifact: `android/libs/fidbek-android.aar` to native SDK `0.2.0`.
- Updated bundled iOS artifact: `ios/FidbekSDK.xcframework` to native SDK `0.2.0`.
- Added native issue frequency selector in the report flow.
- Added device heartbeat ping queue and improved upload resilience.
- Moved token validation to send/submit time.
- Added iOS localization resource bundle (`en`, `tr`) as `ios/FidbekSDKResources.bundle`.
- Bumped package version to `0.2.0`.
