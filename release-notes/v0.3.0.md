# fidbek_flutter v0.3.0

Focused surface release with updated native binaries.

## Highlights

- Updated bundled Android artifact: `android/libs/fidbek-android.aar` to native SDK `0.3.0`.
- Updated bundled iOS artifact: `ios/FidbekSDK.xcframework` to native SDK `0.3.0`.
- Documented the wrapper surface as `configure`, `open`, `identify`, `clearIdentity`, and `shutdown`.
- Native attachment staging helpers remain deprecated and are not exposed through Flutter.
