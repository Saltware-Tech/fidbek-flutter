# fidbek_flutter v0.1.0

## Summary
Initial public release of `fidbek_flutter`.

## Added
- Flutter wrapper plugin for Fidbek mobile SDK.
- MethodChannel API:
  - `configure({required String token, bool shakeToOpenEnabled = true})`
  - `open()`
  - `shutdown()`
- Android integration via bundled `fidbek-android.aar`.
- iOS integration via bundled `FidbekSDK.xcframework`.
- Public package documentation and publishing guide.

## Platform Requirements
- Flutter >= 3.22.0
- Dart >= 3.3.0 < 4.0.0
- Android minSdk 24
- iOS 13+
